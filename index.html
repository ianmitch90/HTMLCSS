Overview:

Objective:

This is a project to show my ability to make diffrent types of layouts on the web.

Style of Organiziation:

I have chosen to demo CSS Grids, Flex Box, Bootstrap, and Traditional methods of trying to organize content for the browser in a responsive way.

Design Choices:

other than the Bootstrap example, my goal is to showcase a media rich experience to liven up an otherwise straight forward excercise.

About

This project primarily a resource for those starting out in developement who want to explore diffrent approaches to web design, with HTML and CSS only. In this example I use a lot of "on hover effects".
I am aware there are tons of css methods, however here im focusing on only one effect and representing that same effect in diffrent forms. I hope you enjoy and take your time and explore the resources. click my name for the CODE.
